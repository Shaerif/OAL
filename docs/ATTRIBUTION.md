## ‚úÖ Attribution Checklist

Use this checklist to ensure proper attribution:

- [ ] Copyright notice included
- [ ] Author/creator name stated
- [ ] Work title mentioned
- [ ] License name (OAL 1.0) specified
- [ ] Link to original work provided (if applicable)
- [ ] Attribution is visible and accessible
- [ ] For derivatives: changes clearly documented

---

## ‚ùå Common Mistakes to Avoid

1. **Incomplete Attribution**: Missing copyright year or author name
2. **Wrong License**: Stating a different license than OAL 1.0
3. **Hidden Attribution**: Placing attribution where users cannot find it
4. **Missing Derivative Notice**: Not documenting changes made
5. **No Link**: Failing to provide link to original work when available
